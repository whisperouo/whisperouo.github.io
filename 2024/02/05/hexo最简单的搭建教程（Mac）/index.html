hexo最简单的搭建教程（基于Mac）

发表于 2024-02-05

第一步

下载并安装nodejs：https://nodejs.org/en

下载完成后会得到两个东西

node.js是本体
npm是包管理器，一会用于下载hexo框架

安装完成后输入命令
node -v
npm -v
如果正常输出则说明二者均已成功安装

第二步

终端输入指令：
npm install -g hexo-cli

-g应该是全局安装的意思

输入
hexo -v
看看是否正常输出

第三步

找个地方建空文件夹

打开终端

进入管理员模式：sudo su

hexo初始化：hexo init

hexo启动：hexo s

此时已经可以在本地运行hexo

第四步

使用Github pages功能进行远程部署hexo博客（其他人输入网址后即可看到我的博客）

注意：github仓库名格式为（github用户名.github.io）

  1. 先将我的github账号与本机进行ssh链接
  2. 修改blog目录下的_config.xml文件中的deploy选项，如下图所示

repo为github仓库中ssh链接

Tips：

hexo常用命令：

hexo clean 清除缓存

hexo g 生成网页

hexo s 启动hexo

hexo d 远程分发hexo

hexo new 新建一篇博客
